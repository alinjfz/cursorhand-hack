The crew

Four agents.
One payday.

01

Hunter

npm run hunt

Scans CSV imports or OpenStreetMap for London businesses with a phone number and no website. Deduplicates and loads leads into Supabase.

  • CSV
  • Overpass
  • Supabase
02

Builder

npm run build

Generates a polished single-page site via Manus or OpenRouter, then deploys to Vercel. Marks leads SITE_READY with a live URL.

  • Manus
  • OpenRouter
  • Vercel
03

Outreach

npm run approach

Sends WhatsApp messages through Wassist — approved templates or agent session messages — with the site preview link.

  • Wassist
  • WhatsApp
  • Templates
04

Payday

/api/wassist

Listens for YES replies on WhatsApp, creates a PayPal sandbox checkout, and sends the payment link. Marks leads PAID on capture.

  • BYOA webhook
  • PayPal
  • Auto-reply
